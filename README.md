## 🚀 Overview

Cavos bridges the gap between retail users and DeFi on Starknet by offering:

- 🔐 Smart contract-based wallets
- 💵 Stablecoin onboarding
- 📈 Curated DeFi investment opportunities
- ⚡ Gasless & seamless user experience
- 🌐 Layer 2 scalability via Starknet

---


## 🔑 Key Features

Cavos delivers a DeFi experience optimized for mobile users:

- 🪪 **Smart Wallet Provisioning** – Wallets are created automatically for users upon onboarding using Starknet account abstraction.
- 💰 **Stablecoin Funding** – Easily deposit USDC and other stablecoins into your wallet.
- 📊 **One-Tap DeFi Investments** – Access curated DeFi protocols with simplified interaction flows.
- 🔄 **Seamless Token Swaps** – Swap supported assets using integrated bridges and routers.
- 📱 **Intuitive Interface** – Mobile-first design built with React Native.
- 🔐 **Secure By Default** – Leverages smart contract wallets for advanced security.
- ⚙️ **Gasless Transactions** – Abstracted gas mechanics using Paymasters for frictionless UX.
- 🌉 **Cross-Layer Connectivity** – Asset bridging between Ethereum and Starknet made simple.

---

## 🛠 Tech Stack

- **Frontend**: React Native (Expo)
- **Smart Contracts**: Cairo (Starknet)
- **Layer 2**: Starknet zkRollup
- **Wallet Infrastructure**: Argent X, Braavos, & custom smart wallet factory
- **DeFi Integration**: Jediswap, Nostra, MySwap
- **Authentication**: Starknet native auth abstraction
- **Transaction Relayers**: Bundlers + Paymasters (ERC-4337 pattern)
